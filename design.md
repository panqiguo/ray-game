

# 乌鸦的去向 — Demo 设计文档 v2

---

## 一、设计目标

一款以牌库表达心理状态的 noir 侦探游戏。玩家在有限天数内整备自己，在高压任务中用有限手牌和可承受的代价推进真相。

**Demo 验证四个问题：**
1. 手牌匹配与非匹配选择是否产生有意义的决策
2. 牌库污染是否形成可感知的情绪曲线
3. 城市整备是否真实影响任务表现
4. 成长系统是否让玩家逐步塑造自己的解题方式

**设计原则：**
- 牌库即潜意识，手牌即当下心境——这是核心隐喻，不可妥协
- 系统收敛：少量资源、一个消耗品、一个完整任务
- 成长改变打法，不只改变数字

## 二、卡牌系统

### 2.1 核心隐喻

牌库 = 你的潜意识。你无法完全控制自己今天的状态。
手牌 = 今天的心境。你只能用手头的状态去应对眼前的事。

### 2.2 花色

| 花色 | 含义 | 典型行动 |
|------|------|----------|
| 理性 | 分析、破解、推演 | 开锁、解码、找漏洞 |
| 直觉 | 观察、搜索、预判 | 搜查、跟踪、察觉异常 |
| 共情 | 安抚、套话、说服 | 谈判、安抚、获取信任 |
| 强硬 | 威吓、硬闯、忍痛 | 破门、对峙、承受伤害 |

### 2.3 起始牌库

**12 张，全部为 0 点基础牌：**

| 花色 | 数量 | 点数 |
|------|------|------|
| 理性 | 3 | 0 |
| 直觉 | 3 | 0 |
| 共情 | 3 | 0 |
| 强硬 | 3 | 0 |

牌库小意味着：
- 每张新增/升级牌的影响都可感知
- 污染牌的稀释效应更弱，压力更真实
- 构筑感更强

### 2.4 状态牌

负面状态牌会被塞入牌库，占据手牌位，无法正常使用。

| 类型 | 例子 | 来源 |
|------|------|------|
| 物理负面 | 流血、咬伤 | Health 伤害 |
| 心理负面 | 疲惫、惊悸 | Stress 过载、硬撑、特定事件 |

正面状态牌本版不做体系，仅保留极少量临时效果。

### 2.5 抽牌循环

```
抽牌 → 出牌/弃牌 → 牌库抽空 → 弃牌堆重洗为新牌库 → 继续
```

- **城市模式：** 一次抽牌 = 一天的精力，抽 5 张
- **任务模式：** 一次抽牌 = 一口气，抽 5 张；抽空后可硬撑

## 三、行动判定

### 3.1 行动结构

每个行动点提供 1–3 种行动方式。每种方式定义：

```
行动方式: 破解门锁
├── 契合花色: 理性
├── 难度: 2
├── 风险: 中
├── 成功: 安静进入，无人察觉
├── 代价成功: 进入，但触发警报（推进 heat 钟 +1）
└── 失败: 门锁损坏，必须绕路
```

## 3.2 判定系统：行动值 + 1d6

### 核心公式

```
行动值 = 卡牌点数 + 契合加成(+1) + 道具/成长加成
```

### 判定流程

1. 选择行动方式
2. 打出卡牌，计算行动值
3. 掷 1d6
4. 根据行动值查表，骰子结果落入哪个区间即为最终结果

### 判定表

行动值决定骰子每个面的含义。**每 +1 行动值，恰好将一个骰面从坏结果翻转为好结果。**

```
        骰面:    ①  ②  ③  ④  ⑤  ⑥
        
行动值 1 (绝境)  失  失  失  代  代  成    失败 50% │ 代价 33% │ 成功 17%
行动值 2 (冒险)  失  失  代  代  代  成    失败 33% │ 代价 50% │ 成功 17%
行动值 3 (对等)  失  失  代  代  成  成    失败 33% │ 代价 33% │ 成功 33%
行动值 4 (优势)  失  代  代  成  成  成    失败 17% │ 代价 33% │ 成功 50%
行动值 5 (掌控)  代  代  成  成  成  成    失败  0% │ 代价 33% │ 成功 67%
行动值 6 (碾压)  成  成  成  成  成  成    失败  0% │ 代价  0% │ 成功 100%
```

> **失** = 失败并付代价
> **代** = 成功但付代价
> **成** = 完全成功

### 每一点的价值

```
+0 → +1:  一个失败面 → 代价面    (少一次彻底失败)
+1 → +2:  一个失败面 → 代价面    (继续收窄失败)
+2 → +3:  一个代价面 → 成功面    (开始出现干净成功)
+3 → +4:  一个失败面 → 成功面    (失败可能性大幅下降)
+4 → +5:  最后的失败面 → 代价面  (不可能再彻底失败)
+5 → +6:  最后的代价面 → 成功面  (完美掌控)
```

玩家每获得 +1，都能在 UI 上直接看到骰面颜色的变化——一格红变黄，或一格黄变绿。这就是"每一点都有清晰感知"。

### UI 表现

选择行动方式并拖入卡牌后，行动面板实时显示：

```
┌─────────────────────────────────────────┐
│  破解门锁                    [理性] 难度 │
│  "老式挂锁，需要耐心和巧劲"             │
│                                         │
│  你的行动值: 2                          │
│  ┌───┬───┬───┬───┬───┬───┐             │
│  │ ✕ │ ✕ │ △ │ △ │ △ │ ✓ │             │
│  │红 │红 │黄 │黄 │黄 │绿 │             │
│  └───┴───┴───┴───┴───┴───┘             │
│  失败 33%  代价 50%  成功 17%           │
│                                         │
│  若使用契合花色 [理性]: 行动值 → 3      │
│  ┌───┬───┬───┬───┬───┬───┐             │
│  │ ✕ │ ✕ │ △ │ △ │ ✓ │ ✓ │             │
│  │红 │红 │黄 │黄 │绿 │绿 │             │
│  └───┴───┴───┴───┴───┴───┘             │
│  失败 33%  代价 33%  成功 33%           │

```

玩家拖入不同卡牌时，骰面颜色实时变化。换一张契合牌，看到一格黄变绿；加一个道具加成，又一格红变黄。**决策的价值肉眼可见。**

### 与风险的配合

判定表决定**成不成**。风险等级决定**失败和代价具体多痛**：

| | 失败（✕）时 | 代价（△）时 |
|------|------------|------------|
| 低风险 | 丢时间，小额损失 | 推进钟 +1 |
| 中风险 | Stress +2，塞入 1 张负面牌 | Stress +1 或推进钟 +1 |
| 高风险 | Health -1，推进钟 +2 | Health -1 或塞入负面牌 |

两个轴完全正交：
- 行动值高 + 高风险 = 大概率成功，但万一出事就很痛
- 行动值低 + 低风险 = 经常失败，但失败了也不致命

### 掷骰的演出

建议骰子不做 3D 物理模拟，而是用**转盘/轮盘风格**：

```
   ┌───┬───┬───┬───┬───┬───┐
   │ ✕ │ ✕ │ △ │ △ │ ✓ │ ✓ │
   └───┴───┴───┴───┴───┴───┘
                 ▲
              [指针停在此处]
              
         结果: △ 成功，但付代价
```

六格横条快速滚动后停下，指针指向最终结果。视觉上直接复用行动面板里已经展示过的那排骰面格，玩家不需要重新理解任何信息。

### 边界情况

- **行动值 0 或负数：** 视为行动值 1（绝境），仍有 17% 奇迹机会
- **行动值 > 6：** 视为行动值 6（碾压），自动成功
- **负面状态牌打出：** 行动值固定为 0（即按行动值 1 处理），代表用创伤驱动行动——几乎必然失败或付代价，但不是完全不可能

### 3.3 风险的职责

风险不决定成不成，只决定**代价多重**：

| 风险等级 | 代价范围 |
|----------|----------|
| 低 | 丢时间、小额金钱、推进钟 +1 |
| 中 | Stress +1~2，或塞入 1 张负面牌 |
| 高 | Health -1，显著推进钟，封闭路径 |

### 3.4 非匹配的意义

非匹配不是一种独立机制，而是一种**自然发生的状态**——当你手里没有契合牌，或者你选择把契合牌留给更关键的行动时，你就在用不对口的心境处理问题。

这不只是数值上少 +1。不同花色打同一个行动点，叙事结果不同：

> **行动点：恶犬挡路**
> - 用【共情】→ 安抚后绕行，安静但耗时
> - 用【强硬】→ 硬闯，受伤但直接
> - 用【理性】→ 观察狗链弱点，拽断后通过，推进 heat 钟
> - 用【直觉】→ 察觉狗的巡逻间隙，等待后溜过

## 四、资源系统

### 4.1 四项核心状态

```
╔══════════════════════════════════════════════════════════╗
║  ♥ Health: 5/5  │  ◆ Stress: 0/8  │  $ Money: 30  ║
╚══════════════════════════════════════════════════════════╝
```

| 资源 | 含义 | 归零/满载后果 |
|------|------|---------------|
| Health | 身体承受能力 | 归零 = 任务失败 |
| Stress | 精神压力 | 满载 = 塞入心理负面牌，可能触发后遗症 |
| Money | 通用整备资源 | 无钱 = 无法购买/打点 |

### 4.2 进度钟

进度钟是通用的计时/计量工具，借鉴 Blades in the Dark。任何需要追踪渐进变化的东西都可以用进度钟表示。

```
Heat 钟 [●●●○○○] 3/6
"黑帮已经注意到有人在打听屠宰场的事"

满钟效果: 黑帮全面警戒，任务中增援出现，部分路径封锁
```

**进度钟的规则：**
- 可以在任何场景创建
- 可增可减（取决于玩家行动）
- 满/空时触发预定义事件
- 不同钟互相独立

**Demo 中使用的进度钟：**

| 钟名 | 格数 | 触发条件 | 满钟效果 |
|------|------|----------|----------|
| Heat | 6 | 暴力行动、暴露、被发现 | 任务中增援、封路、NPC 态度恶化 |
| 乌鸦的时间 | 6 | 每过一天 -1，特定行动可冻结 | 归零 = 乌鸦被处决，任务结局改变 |
| 屠宰场警报 | 4 | 任务内噪音、被发现 | 满钟 = 头目转移乌鸦，必须追击 |

### 4.3 消耗品：烟卷

烟卷是本版唯一的消耗品。

| 场景 | 效果 | 代价 |
|------|------|------|
| 城市 | Stress -2 | 当天结束时塞入 1 张【疲惫】 |
| 任务 | 弃掉 1 张手中的负面牌，补抽 1 张 | Stress +1 |

> *你靠在墙边点了一根。烟雾散去，手不抖了。*

### 4.4 污染清理途径

牌库污染不能只增不减，否则系统会进入死亡螺旋。

| 方式 | 位置 | 效果 | 代价 |
|------|------|------|------|
| 包扎 | 城市·诊所 | 移除 1 张物理负面牌 | Money -15，花费 1 天行动 |
| 买醉 | 城市·酒吧 | 移除至多 2 张任意负面牌 | 花费 1 天行动，次日抽牌 -1（宿醉） |
| 整理思绪 | 城市·事务所 | 移除 1 张心理负面牌 | 花费 1 天行动（无其他行动） |
| 烟卷 | 任意 | 弃掉手中 1 张负面牌 | 见上方烟卷效果 |

**设计意图：** 清理总是需要时间或金钱，形成与整备行动的竞争关系。买醉效率最高但有宿醉惩罚。

## 五、双模式结构

### 5.1 城市整备（Downtime）

```
┌─────────────────────────────────────────────────────┐
│                    城 区 地 图                        │
│                                                     │
│    [事务所]          [红灯区]         [码头]         │
│       │                │               │            │
│       └── 办公桌       ├── 暗巷        ├── 集装箱区  │
│           案卷         │   混混        │   打工      │
│                        └── 酒吧        └── 走私商    │
│                                                     │
│    [黑市]             [诊所]                         │
│       │                │                            │
│       └── 街边摊       └── 包扎                      │
│           烟卷/道具        清理物理负面               │
│                                                     │
└─────────────────────────────────────────────────────┘
```

**每天流程：**
1. 抽 5 张手牌（宿醉则 4 张）
2. 在地图上选择场所，用手牌执行行动
3. 手牌打空或主动点击【结束今天】
4. 推进"乌鸦的时间"钟 -1
5. 若已解锁"记案本"，可保留 1 张牌

**地图交互只有两层：**
- 城区层 → 点击展开场所层
- 场所层 → 点击行动点，弹出行动面板
- 不做第三层

### 5.2 任务推进（Score）

```
┌─────────────────────────────────────────────────────┐
│                屠 宰 场 蓝 图                        │
│                                                     │
│  ┌──────────┐    ┌──────────┐    ┌──────────┐      │
│  │  外 围   │───→│  走 廊   │───→│  冷 库   │      │
│  │          │    │          │    │          │      │
│  │·铁丝网   │    │·恶犬     │    │·证据台   │      │
│  │·巡逻守卫 │    │·灯控室   │    │·头目     │      │
│  │          │    │          │    │·乌鸦     │      │
│  └──────────┘    └──────────┘    └──────────┘      │
│                                                     │
│  屠宰场警报 [○○○○] 0/4                              │
└─────────────────────────────────────────────────────┘
```

**任务规则：**
- 进入后城市节点隐去，不可返回
- 手牌打空后可选择**硬撑**
- 区域内行动点可按任意顺序处理
- 区域间单向推进（外围 → 走廊 → 冷库）

### 5.3 硬撑

| 硬撑次数 | 抽牌数 | Stress 代价 |
|----------|--------|-------------|
| 第 1 次 | 3 张 | +2 |
| 第 2 次 | 2 张 | +3 |
| 第 3 次 | 1 张 | +4 |

每次硬撑后，若 Stress 满载，立即塞入 1 张心理负面牌并重置 Stress 为上限 -2。

## 六、成长系统

### 6.1 原则

准备、保留、升级、闪回**全部是成长解锁项**，不是默认能力。玩家通过成长逐步塑造牌组和解题方式。

### 6.2 Demo 成长节点

成长点通过完成城市/任务中的关键行动获取（不做经验值系统）。

**生存向：**

| 节点 | 效果 |
|------|------|
| 冷静习惯 | Stress 上限 +2 |
| 老伤钝化 | 每次任务中首次受 Health 伤害时，不塞入物理负面牌 |

**构筑向：**

| 节点 | 效果 |
|------|------|
| 记案本 | 每天结束时可保留 1 张手牌到次日 |
| 预先踩点 | 进入任务前，检视牌库顶 3 张，可将其中 1 张置底 |
| 偏执专长 | 选择 1 种花色，该花色 1 张基础牌永久升级为 1 点 |

**技巧向：**

| 节点 | 效果 |
|------|------|
| 闪回 | 任务中消耗 4 Stress，将 1 张手牌临时视为任意花色打出 |

### 6.3 闪回说明

闪回是高戏剧性的逆转能力。

- 消耗 4 Stress
- 选择手中 1 张牌，宣告它临时变为任意花色
- 触发一段闪回叙事片段（"三天前你在酒吧听到的那句话突然有了意义——"）
- 本次行动按宣告的花色结算

## 七、Demo 流程：乌鸦的去向

### 序幕：事务所醒来

> 阴雨天。桌上放着乌鸦寄来的求救信。

**教学内容：**
- 抽取首日 5 张手牌
- 介绍四花色
- 介绍 Health / Stress / Money
- 展示进度钟：Heat [○○○○○○] 0/6，乌鸦的时间 [●●●●●●] 6/6

**侦探独白系统首次触发：**
> *（根据手牌构成生成）*
> *手里全是强硬："今天脑子里只有拳头。"*
> *花色均衡："难得清醒的一天。好好利用。"*

---

### 第 1 天：熟悉城市

**目标：** 教学地图缩放、行动判定、赚钱、买烟卷

**可选行动：**

| 地点 | 行动 | 契合花色 | 难度 | 结果 |
|------|------|----------|------|------|
| 码头·集装箱区 | 搬货打工 | 强硬 | 1 | Money +15 |
| 红灯区·暗巷 | 堵路混混·套话 | 共情 | 2 | 获得线索碎片 A |
| 红灯区·暗巷 | 堵路混混·威吓 | 强硬 | 2 | 获得线索碎片 A，Heat 钟 +1 |
| 黑市·街边摊 | 买烟卷 | — | — | Money -10，获得烟卷 ×2 |

**天结束：** 乌鸦的时间 -1 → [●●●●●○] 5/6

---

### 第 2 天：深入调查

**目标：** 教学污染清理、进度钟的增减、信息收集

**新增/变化行动：**

| 地点 | 行动 | 契合花色 | 难度 | 结果 |
|------|------|----------|------|------|
| 红灯区·酒吧 | 找线人喝酒 | 共情 | 2 | 获得线索碎片 B（屠宰场布局），Heat 钟 +1 |
| 红灯区·酒吧 | 买醉 | — | — | 移除至多 2 张负面牌，次日抽牌 -1 |
| 码头·走私商 | 打听屠宰场 | 直觉 | 3 | 获得线索碎片 C（守卫换班时间），可冻结乌鸦时间钟 1 天 |
| 诊所 | 包扎 | — | — | Money -15，移除 1 张物理负面牌 |
| 事务所 | 整理思绪 | — | — | 移除 1 张心理负面牌（占据整天行动） |

**线索碎片的作用：**
- 碎片 A（混混情报）：确认乌鸦在屠宰场 → 解锁任务入口
- 碎片 B（屠宰场布局）：任务中走廊区域额外显示一个隐藏行动点
- 碎片 C（换班时间）：任务外围区域守卫难度 -1

**天结束：** 乌鸦的时间 -1 → [●●●●○○] 4/6

### 第 3 天：最后准备或直接出发

**玩家选择：**
- 继续整备（买烟卷、清理污染、赚钱、解锁成长节点）
- 直接进入屠宰场任务

**压力来源：** 乌鸦的时间钟还剩 3–4 格。每多等一天就少一格。如果归零，任务仍可进入，但乌鸦已死，结局分支改变。

**天结束（若未进入任务）：** 乌鸦的时间 -1

### 结局结算

根据以下变量组合决定结局：

| 变量 | 状态 |
|------|------|
| 乌鸦是否存活 | 时间钟是否归零 |
| 是否获得账本 | 证据台是否成功 |
| 是否与乌鸦对话 | 是否执行对话行动 |
| 头目处置方式 | 对峙/谈判/绕过 |
| 玩家自身状态 | Health/Stress/负面牌数量 |

**Demo 结局示例：**

| 结局 | 条件 |
|------|------|
| 真相与代价 | 救出乌鸦 + 获得账本 + 对话，但 Health/Stress 严重 |
| 沉默的正义 | 救出乌鸦，未获得真相，头目逃脱 |
| 迟来一步 | 乌鸦时间钟归零，到达时只剩遗体和线索 |
| 交易 | 用账本换人，真相被掩埋，Heat 飙升 |
| 倒在冷库 | Health 归零，任务失败 |
