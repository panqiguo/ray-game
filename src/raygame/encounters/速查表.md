Encounter Scheme 速查表

顶层：
(include "file.scm")
(define name expr)
(meta ...)
(state ...)
(reacts ...)
(on-success (list effect...))
(on-fail (list effect...))
最后一个表达式必须返回 scene

meta：
(meta :key 'id :title "标题" :desc "描述")
必填：:key, :title
选填：:desc，默认 ""

state：
(state
  (field initial)
  (clock-field (clock ...)))

clock：
(clock :title "名称" :initial 0 :max 4 :persist 'encounter)
必填：:title, :initial, :max
选填：:persist，默认 "encounter"

scene：
(scene
  :key 'scene_key
  :title "标题"
  :desc "描述"
  :show-clocks (list clock-a clock-b)
  :actions (list action-a action-b)
  :children (list scene-a scene-b))
必填：:title
选填：:key, :desc, :show-clocks, :actions, :children
默认：:desc "", :show-clocks (), :actions (), :children ()

action：
(action
  :key 'action_key
  :title "标题"
  :desc "描述"
  :inputs (list input...)
  :before (list effect...)
  :effects (list effect...)
  :check check-expr)
必填：:title
选填：:key, :desc, :inputs, :before, :effects, :check
规则：:check 和 :effects 互斥
规则：:before 可和 :check 共存，也可和 :effects 共存
规则：有 :check 时，:before 是检定前效果；无 :check 时，:before + :effects 都会直接执行

check：
(check
  :suits (list force reason)
  :risk mid
  :ok outcome
  :partial outcome
  :fail outcome)
必填：:suits, :risk, :ok, :partial, :fail
:risk 可用：low, mid, high
:suits 可用：reason, force, empathy, instinct

outcome：
(outcome "结果文本" (list effect...))

reacts：
(reacts
  (react :key 'some_key :when condition :then (list effect...)))
react 必填：:when, :then
react 选填：:key

inputs：
(resource key amount)
(resource key amount label)

(item key)
(item key amount)
(item key amount label)
(item key amount label consume)

(card key)
(card key label)

input 示例：
(resource 'money 20 "现金")
(item 'lockpick 1 "开锁器" true)
(card any)
(card negative "负面牌")

effects：
(effect 'clock+ clock-field amount)
(effect 'clock- clock-field amount)
(effect 'set field value)
(effect 'health amount)
(effect 'stress amount)
(effect 'resource resource-key amount)
(effect 'finish outcome)
(effect 'start-dialogue dialogue-id)
(effect 'start-quick-dialogue "对白内容")
(effect 'reset-hand)

effect 示例：
(effect 'clock+ alert 1)
(effect 'clock- enemy_hp 2)
(effect 'set phase 'entry)
(effect 'set opened true)
(effect 'health -1)
(effect 'stress 2)
(effect 'resource 'money 80)
(effect 'finish 'success)
(effect 'finish 'fail)
(effect 'finish 'abort)
(effect 'start-dialogue "挣脱束缚")
(effect 'start-quick-dialogue "# 标题\n第一段\n\n第二段")
(effect 'reset-hand)
