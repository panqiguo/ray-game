结论

你的说法有一定依据，但如果说成“Romero 离开后，id 就不再有真正突破性创新”，这个判断会过于简单。 更公平的说法应该是：

Romero 离开后，id Software 的创新重心明显从“游戏设计 + 技术 + 文化气质的混合爆发”，转向了“引擎技术、图形技术、多人竞技与稳定生产”。 他们仍然有技术突破和局部设计突破，但很少再像 Wolfenstein 3D / Doom / Quake 那样，连续重塑整个游戏行业的基本范式。

所以，你的直觉大体可以成立，但需要加两个限制： 第一，不是“没有创新”；第二，不能把早期成功完全归因于 Carmack + Romero 的性格冲突。

1. 事实时间线：Romero 离开后的 id 并没有停摆

John Romero 是在 Quake 之后、1996 年离开 id Software。资料通常把原因描述为他与 John Carmack 在公司方向、开发节奏和 Quake 制作方式上的冲突：Romero 倾向更强的愿景驱动和游戏体验扩张，Carmack 更强调项目推进、技术纪律和可完成性。:contentReference[oaicite:0]{index=0}

Romero 离开后，id 仍然做出了这些重要作品：

时期 作品 主要变化 是否算突破
1997 Quake II 更完整的科幻军事 FPS 结构，技术迭代，商业成功 中等；更像强化，不像范式革命
1999 Quake III Arena 放弃传统单人战役，集中做竞技多人 FPS 有突破性，尤其在竞技 FPS、网络对战、引擎授权方面
2004 Doom 3 id Tech 4，统一动态光照、阴影体、法线贴图等 技术突破明显；玩法上较保守
2011 Rage id Tech 5、MegaTexture/虚拟纹理、多平台生产、载具与开放区域 技术上有野心；游戏设计影响力有限
2016 / 2020 Doom / Doom Eternal “push-forward combat”，通过处决、资源循环、近距离压迫重塑现代 Doom 设计突破较强，但这是后 Carmack 时代的 id

Quake III Arena 本身就是一个反例：它几乎把传统单人内容拿掉，把整个游戏押注在竞技多人上；当时 Wired 报道也提到，Carmack 认为取消传统单人模式可以让多人部分更快、更顺。:contentReference[oaicite:1]{index=1}

2. 你的观点为什么有依据

2.1 早期 id 的创新确实是“技术狂人 + 体验狂人”的组合

早期 id 的厉害之处，不只是 Carmack 写出了强大的引擎，也不只是 Romero 会做关卡、会推广、会把游戏做得爽。真正的化学反应在于：

  • Carmack 把 PC 硬件推到极限：平滑卷轴、伪 3D、BSP、网络对战、真 3D 引擎。
  • Romero 把这些技术变成可感知的玩家体验：速度、暴力反馈、关卡节奏、秘密区域、玩家自由感、摇滚式气质。
  • Adrian Carmack、Tom Hall、Sandy Petersen、American McGee 等人 也贡献了美术、世界观、关卡、怪物、氛围。

所以，说 Doom 和 Quake 是“Carmack 的技术 + Romero 式游戏感 + 团队美术/关卡/文化气质”的结果，比单说“Carmack 做的”或“Romero 做的”都更准确。Masters of Doom 这类资料也通常把二人的互补和冲突作为 id 早期神话的核心。:contentReference[oaicite:2]{index=2}

2.2 Romero 离开后，id 的作品确实更像“工程导向公司”

Romero 离开后，id 的强项仍然很清楚：引擎、性能、图形技术、多人射击手感、跨平台技术、工具链。但它较少再提出新的“游戏生活方式”或新的“想象空间”。

例如 Doom 3 的 id Tech 4 有明显技术创新：统一动态光照、实时阴影、法线贴图、逐像素光照，这些在当时很激进。:contentReference[oaicite:3]{index=3} 但从玩法感受上，它更像是把 Doom 改造成黑暗走廊恐怖 FPS，而不是像 1993 年 Doom 那样重新定义“电脑游戏可以是什么”。

Rage 也类似：id Tech 5 的虚拟纹理/MegaTexture 很有技术野心，游戏本身也尝试了载具、开放区域、RPG 式任务结构。:contentReference[oaicite:4]{index=4} 但它没有成为行业设计范式的核心参照，影响力远不如 Doom、Quake 或后来的 Half-Life、Halo、Call of Duty 4。

2.3 “稳定可靠但不再极端创新”这个描述大体合理

id 在 Romero 离开后没有失败。它仍是顶级 FPS 技术公司，也长期保持品牌影响力。但从“连续改变行业”的角度看，确实明显降速。

早期 id 几乎每隔一两年就推出一次范式级变化：

  • Commander Keen：PC 平滑卷轴平台游戏。
  • Wolfenstein 3D：现代 FPS 的重要奠基。
  • Doom：网络死亡竞赛、Mod 文化、共享软件发行、暴力快节奏 FPS 的爆炸。
  • Quake：真 3D、客户端/服务器网络、竞技 FPS、引擎授权生态。

Romero 之后，id 仍有重要作品，但更常见的是“把既有 FPS 传统做到很强”，而不是“创造一个全新的游戏时代”。

3. 反驳：这个观点不能说得太绝对

3.1 Quake III Arena 本身就是一次重大设计选择

如果说 Romero 离开后 id 完全没有突破，那 Quake III Arena 很难解释。它把单人剧情几乎彻底边缘化，直接把“竞技场多人 FPS”作为产品核心。这在今天看好像自然，但在 1999 年是很激进的商业选择。它同时推动了电子竞技、网络对战、引擎授权和竞技地图设计。:contentReference[oaicite:5]{index=5}

所以更准确的说法是：Romero 离开后，id 的创新从“世界、关卡、玩法幻想的综合创新”，变成了“技术平台 + 竞技系统 + 图形表现”的创新。

3.2 Doom 3 的技术突破不能低估

Doom 3 的玩法评价可以争论，但 id Tech 4 的技术路线确实重要。全动态光照、统一阴影、法线贴图这些东西让它在图形技术史上有位置。:contentReference[oaicite:6]{index=6}

换句话说，id 没有停止创新，只是它的创新越来越偏向“图形引擎论文式突破”，而不是“玩家一上手就觉得游戏规则被重新发明了”。

3.3 Romero 离开后自己也没有证明“没有他不行”

这是反驳你观点里最重要的一点。Romero 离开 id 后创立 Ion Storm，最著名的项目是 Daikatana，但它通常被视为高调失败案例。也就是说，如果把早期 id 的成功主要归因于 Romero 的创意气质，那么 Romero 离开后的成绩并不能强力支持这个说法。

这说明早期 id 的成功不是“Romero 单独代表创新，Carmack 单独代表保守”。更可能是：

  • Romero 需要 Carmack 的技术现实感来约束愿景。
  • Carmack 需要 Romero、Hall、Petersen、McGee、Adrian Carmack 等人把技术转化成玩家欲望。
  • 早期 id 还赶上了 PC 硬件、共享软件、Mod、局域网/互联网、3D 加速卡等历史窗口。

3.4 不是所有“冲突”都会产生创新

你说“性格跟理念有冲突的人，但是有才华的人在一起做出了真正好的游戏和创新”，这个判断有解释力，但也容易浪漫化。

冲突只有在几个条件下才会变成创造力：

  • 双方能力都足够强。
  • 冲突围绕作品本身，而不是纯粹权力斗争。
  • 有足够短的反馈循环，能快速验证谁的想法有效。
  • 项目规模还没有大到必须流程化管理。
  • 公司文化允许冒险，但还没有被冒险拖垮。

id 早期满足这些条件。Ion Storm 某种程度上也有“愿景”和“明星气质”，但缺少足够有效的生产约束，结果就不是创新，而是失控。

4. 更公平的最终判断

你的观点可以改写成下面这个版本，会更有依据：

id Software 早期最强的创新，来自 Carmack 的极端技术能力、Romero 的玩家体验直觉、Adrian Carmack 的视觉风格，以及其他设计师共同形成的粗粝、快速、反商业常规的团队化学反应。 Romero 离开后，id 没有失去技术能力，也没有停止做优秀 FPS，但它的创新重心明显收窄，更偏向引擎、图形、性能和竞技多人，而较少再产生像 Doom 或 Quake 那样重塑整个游戏媒介的作品。

这个说法比“Romero 一走，id 就不创新了”更准确；也比“Carmack 才是一切，Romero 不重要”更公平。

5. 对你的游戏创作有什么启发

如果把这个问题转回你自己的游戏开发，它的启发不是“必须找一个和自己冲突的人”，而是：

  • 技术理性会让项目做得出来。
  • 体验冲动会让项目有生命力。
  • 美术/叙事气质会让项目被记住。
  • 生产约束会防止愿景变成空想。

对 id 来说，Carmack 像“能让世界运行的物理法则”，Romero 像“让玩家想住进这个世界的欲望”。早期 id 的奇迹，可能正是这两种力量短暂地处在同一个房间里。
